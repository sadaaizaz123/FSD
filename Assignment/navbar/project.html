🚀 Career Compass

Smart Career & Course Recommender

🎯 Core Idea

Career Compass is a smart web application where students can enter their interests and skills, and instantly get recommendations for:

  • Best career paths (Engineer, Designer, Data Scientist, etc.)
  • Matching online courses (YouTube, Coursera, Udemy, etc.)
  • Skill gap analysis with progress visualization

🔑 Key Features

📋 Interest Quiz

Students pick skills like Coding, Art, Math, Public Speaking. Fun MCQ quiz option too!

💼 Career Recommendations

JS maps skills → careers. Example: Coding + Math → Software Engineer.

📚 Course Suggestions

3–5 recommended courses per career (hardcoded JSON or free APIs).

📊 Skill Gap Analysis

Shows missing skills (e.g., HTML + CSS but missing JS). Visualized via Chart.js.

💾 Save My Plan

User’s career path + courses saved in Local Storage for later viewing.

🌙 UI Extras

Dark/Light mode, smooth animations, dashboard-style results page.

🛠️ Tech Stack

  • Frontend: HTML, CSS, Bootstrap
  • Logic: Vanilla JavaScript (arrays, objects, mapping)
  • Charts: Chart.js for skill gap visualization
  • Data: JSON dataset of careers & courses
  • Storage: LocalStorage for saving user plans

🖼️ UI Flow

  1. Homepage → Welcome + Start Quiz button
  2. Quiz Page → Select skills/interests
  3. Results Page → Career Recommendation + Skill Gap Chart + Course Suggestions
  4. My Plan Page → Saved career path + courses

⚡ Why It Will Impress

  • Feels AI-powered but built with simple JavaScript mapping.
  • Not a basic app — it combines career + course + visualization.
  • Useful for students → professors appreciate practical utility.
  • Charts + modern UI → instant WOW factor.
